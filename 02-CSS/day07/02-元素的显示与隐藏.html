---------display-------------- 1.display: none 隐藏对象 2.display: block 除了装换为块级元素之外,同时还有显示元素的意思 特点:隐藏之后,不再保留位置 ---------visibility:可见性-------------- 1.visibility:visible 对象可视 2.visibility:hidden 对象隐藏 特点:隐藏之后,继续保留位置 --------overflow:内容溢出(解决塌陷,消除浮动)----- 1.overflow:visible 不剪切内容也不添加滚动条 2.overflow:hidden 不显示出国对象尺寸的内容,超出部分隐藏 3.overflow:scroll 不管超出内容否,总是显示滚动条 4.overflow:auto 超出自动显示滚动条,不超出不显示
